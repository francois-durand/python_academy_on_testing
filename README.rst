Python Academy on Testing contains examples of testing for the `Python Academy`_ of the `Lincs`_.

* Free software: GNU General Public License v3
* Documentation: https://python-academy-on-testing.readthedocs.io.

.. _`Python Academy`: https://www.lincs.fr/research/working-groups/python-academy/
.. _`Lincs`: https://www.lincs.fr/

Features
--------

* Demonstrate testing in Python (mainly doctest and pytest).

Download
--------

The sources for Python Academy on Testing can be downloaded from the `Github repo`_:

.. code-block:: console

    $ git clone git://github.com/francois-durand/python_academy_on_testing

.. _Github repo: https://github.com/francois-durand/python_academy_on_testing
